Exercices - Fonctions définies par parties

Exercice 1 : Lecture d’une fonction définie par parties

Considère la fonction suivante :

\[ f(x) = \begin{cases} 3x + 4, & \text{si } x < -2 \\ -x + 1, & \text{si } -2 \leq x < 2 \\ 5, & \text{si } x \geq 2 \end{cases} \]

Évalue : \( f(-3) \)

Exercice 2 : Calcul de valeur

Considère la fonction suivante :

\[ g(x) = \begin{cases} 2x - 1, & \text{si } x < 0 \\ 4, & \text{si } x \geq 0 \end{cases} \]

Évalue : \( g(-2) \)

Exercice 3 : Vérification de continuité

La fonction suivante est-elle continue en \( x = 1 \) ?

\[ h(x) = \begin{cases} -2x + 3, & \text{si } x < 1 \\ 1, & \text{si } x \geq 1 \end{cases} \]

Répondez par "Oui" ou "Non".
